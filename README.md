INSTALLATION INSTRUCTIONS : 



ADD THE DEPENDENCIES : 
- Node.js
- Composer
- XAMPP or Laragon (Optional)


FRONT END
- npm install
- npm install @tanstack/react-table

BACK END
- composer install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate
